# AI/ML Internship Projects

This repository contains 3 beginner-friendly AI/ML projects:

## 📊 Task 1: Iris Dataset Visualization
- Dataset: Seaborn Iris dataset
- Techniques:
  - Data exploration
  - Scatter plots
  - Histograms
  - Boxplots

## 📈 Task 2: Stock Price Prediction
- Data Source: Yahoo Finance (AAPL)
- Model: Linear Regression
- Goal: Predict stock closing prices

## ❤️ Task 3: Heart Disease Prediction
- Dataset: Heart Disease dataset
- Model: Logistic Regression
- Evaluation:
  - Accuracy
  - Confusion Matrix
  - ROC Curve
